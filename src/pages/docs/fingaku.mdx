# Fingaku

Fingaku es una aplicacion web para manejar tus finanzas como estudiante. Sabemos que puede ser complicado controlar tu dinero mientras que te enfocas en tus estudios, es por eso que creamos una plataforma en donde de manera rapida y facil puedes ver como optimizar tus gastos, poner objetivos financieros, crear presupuestos y analizar tu dinero.

Para usar Fingaku lo único que necesitas que hacer es crear una cuenta en la página [https://fingaku.kaitakami.dev/signin](https://fingaku.kaitakami.dev/signin). Contamos con dos planes, uno completamente gratuito y acceso a casi todas las funciones menos poder hablar con nuestro chatbot FingakuAI. Para hablar con FingakuAI tienes que contratar el plan Pro con un costo de $1.9 dolares.
