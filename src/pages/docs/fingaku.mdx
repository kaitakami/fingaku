# Fingaku
Fingaku es una aplicación web para manejar tus finanzas como estudiante. Sabemos que puede ser complicado controlar tu dinero mientras te enfocas en tus estudios, por eso creamos una plataforma donde de manera rápida y fácil puedes ver cómo optimizar tus gastos, establecer objetivos financieros, crear presupuestos y analizar tu dinero.

Para usar Fingaku, lo único que necesitas hacer es crear una cuenta en la página https://fingaku.kaitakami.dev/signin. Contamos con dos planes: uno completamente gratuito con acceso a casi todas las funciones, excepto poder hablar con nuestro chatbot FingakuAI. Para hablar con FingakuAI, debes contratar el plan Pro, que tiene un costo de $1.9 dólares.

# FingakuAI
FingakuAI es una funcionalidad de Fingaku que te permite hacer preguntas relacionadas con finanzas. Aún esta en versión beta por lo que es importante verificar la información. También recuerda que no es consejo financiero y debes de tomar tus decisiones bajo tu propia responsabilidad.
